Add `test`, `test:ci` and `test:watch` tasks powered [Mocha][mocha] to your
Sakefile.

## Install
```bash
$ npm install sake-mocha --save-dev
```

## Usage
In your Sakefile:

```coffee
use('sake-mocha')
```
